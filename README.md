# AI Pregnancy Symptom Checker

An intelligent tool for late-stage pregnancy symptom monitoring and risk detection.

## Features
- Symptom risk assessment (Low/Medium/High)
- Baby kick tracker
- Emergency guidelines

## How to Use
1. Select any symptoms you're experiencing
2. Click "Analyze Symptoms" for risk assessment
3. Track baby movements in the kick tracker
4. Follow the risk level recommendations

## ⚠️ Medical Disclaimer
This tool is for educational purposes only. Always consult your healthcare provider for medical advice.
